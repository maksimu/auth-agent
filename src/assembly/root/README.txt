Auth Agent
==========

Distributed directories:

 * bin  - Startup scripts

 * conf - Configuration files

 * lib  - JAR libraries

 * logs - Log and output files

 
RUNNING
=======

To run in Windows

  $ cd bin
  $ auth-agent.bat start

  or

  $ path\to\auth-agent\bin\auth-agent.bat start
  
To run in Linux

  $ cd bin
  $ ./auth-agent start

  or 
  
  $ /path/to/auth-agent/bin/auth-agent start
